Map/Set vs WeakMap/WeakSet
Key / Value type
Map key any type
WeakMap key objects only
WeakMap value anything ✅
WeakSet value objects only
Garbage Collection
Map/Set manual delete needed
WeakMap/WeakSet auto cleaned by GC
Size & Iteration updated
.size Map/Set ✓   Weak ✗
for..of / forEach Map/Set ✓   Weak ✗
.keys() .values() Map/Set ✓   Weak ✗
.clear() Map/Set ✓   Weak ✗
Methods updated
Map set get has delete clear
WeakMap set get has delete
Set add has delete clear
WeakSet add has delete
clear() does what? new
delete(key) removes one item
clear() removes everything at once
Use when
Map/Set general storage, need to loop
Weak data tied to object lifetime
